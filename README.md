This repository is my public learning journal and practical portfolio for Linux.

Instead of simply completing courses, I wanted to develop a deeper understanding of how Linux is used by developers, data analysts, AI/ML engineers, cloud practitioners, and system administrators.

Through this repository, I document:

- 📖 Concepts and notes

- 🛠️ Bash scripting practice
- 🔍 Terminal-based challenges
- ☁️ Linux concepts relevant to Cloud Computing

- 🚀 Mini projects and automation scripts

The goal is simple:

> Learn Linux by building, practicing, documenting, and sharing consistently.

# 🧩 Terminal Challenges

To strengthen problem-solving skills, I regularly solve Linux challenges.

Example:

### Challenge

Find all `.txt` files modified within the last 7 days.

### Solution

```bash
find . -name "*.txt" -mtime -7
```

### Learning Outcome

Learned how to combine filters and file search operations efficiently.
