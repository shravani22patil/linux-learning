## 📚 Commands Learned

### 1. `pwd`

**Purpose:**
Displays the current working directory.

**Syntax:**

```bash
pwd
```

**Example Output:**

```bash
/home/shravani
```

**Use Case:**
Helps identify where you currently are in the file system.

---

### 2. `ls`

**Purpose:**
Lists files and directories.

**Syntax:**

```bash
ls
```

**Useful Variations:**

```bash
ls -l
ls -a
ls -la
```

**Use Cases:**
- View contents of a directory
- Check hidden files
- Display detailed file information

---

### 3. `cd`

**Purpose:**
Changes the current directory.

**Syntax:**

```bash
cd directory_name
```

**Examples:**

```bash
cd Documents
cd ..
cd ~
```

**Use Cases:**
- Navigate between folders
- Return to the home directory
- Move to parent directories

---

### 4. `mkdir`

**Purpose:**
Creates a new directory.

**Syntax:**

```bash
mkdir project_folder
```

**Example:**

```bash
mkdir LinuxPractice
```

**Use Case:**
Organizing files and projects.

---

### 5. `touch`

**Purpose:**
Creates an empty file.

**Syntax:**

```bash
touch filename.txt
```

**Example:**

```bash
touch notes.txt
```

**Use Case:**
Quickly create files from the terminal.
